# greenden-tailwind-css
 
Greenden is a modern and responsive plant e-commerce website built using Tailwind CSS. The website is designed to provide users with a clean, minimal, and nature-inspired interface to explore and purchase indoor and outdoor plants online.

🚀 Live Preview

 https://muthulakshmi-m2004.github.io/Greenden-Tailwind-css/

 

📌 Project Overview

Greenden is a front-end web project that focuses on:
Responsive UI Design
Utility-first CSS styling using Tailwind CSS
Clean layout structure
Modern e-commerce design principles
The website provides a smooth user experience with visually appealing sections and mobile-friendly responsiveness.

🛠️ Built With

HTML5
Tailwind CSS
Responsive Design Principles
Flexbox & Grid Layout

📂 Website Sections

The Greenden website includes the following sections:
🏠 Home

Hero section with banner image and call-to-action button
Highlighted plant categories

🌱 Products

Display of various plants with images
Price details
Add to cart / Buy options

📞 Contact

Contact form
Email and phone details
Social media links
